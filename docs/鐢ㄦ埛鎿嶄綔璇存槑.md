# 用户操作说明

## 普通用户

- “球队”页：搜索球队并进入球队主页；
- 球队主页：查看基本信息、积分、阵容和赛程；
- “球员”页：按姓名、球队、号码、位置筛选；
- “赛程赛果”页：按轮次、球队和状态筛选；
- 比赛详情：查看主客队、自动比分、事件时间线和出场记录；
- “积分榜”“射手榜”“球员统计”页：查看统计结果。

## 管理员

1. 打开后台管理页面；
2. 输入环境变量 `ADMIN_KEY` 对应的管理员密钥；
3. 可维护赛季、赛事、球队、球员和比赛；
4. 比赛管理不提供手工比分输入；
5. 录入普通进球、点球、乌龙球、红黄牌和换人；
6. 进球可选择助攻者，换人必须选择换上球员；
7. 新增、修改、删除事件后，比分、积分榜和球员统计自动刷新；
8. 通过出场记录维护球员是否出场及上下场时间；
9. 数据异常时可点击“全量重算统计”。

## 删除限制

- 有球员或历史比赛的球队不能直接删除；
- 有事件或出场记录的球员不能直接删除；
- 删除比赛会级联删除该场事件和出场记录；
- 删除事件后比分会自动变化。
