# 修订问题清单

本版本针对原项目完成以下修订：

1. 将“日期”统一改为“比赛时间”；
2. 新增球队详情主页；
3. 完善比赛详情，显示主客队、比分、轮次、时间、状态和球员姓名；
4. 增加点球、乌龙球和换人录入与展示；
5. 后台由手输ID改为下拉选择；
6. 增加球队、球员、比赛、事件完整CRUD；
7. 增加赛季、赛事维护接口与后台页面；
8. 增加比赛和球员多条件检索；
9. 恢复主客队不同约束；
10. 增加队内号码唯一和关键字段非空约束；
11. 删除 `MatchEvent.team_id`，解决传递依赖；
12. 比分由事件自动计算，避免双重事实来源；
13. 助攻与进球直接关联，换上与换下直接关联；
14. 新增 `MatchAppearance`，解决出场次数无来源问题；
15. 修改比赛状态、事件和出场记录后同步刷新统计；
16. 增加事件修改接口；
17. 限制历史球队级联删除；
18. 使用最小权限应用数据库账户；
19. 写接口增加管理员密钥；
20. 初始化脚本使用事务并自动生成统计；
21. 页面和Flask合并为单服务，取消开放CORS；
22. 文档明确当前数据只是课程演示子集。
